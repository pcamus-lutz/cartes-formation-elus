# Cartographie des dépenses de formation des élu·es local·es

Site statique regroupant les cartes interactives des dépenses exécutées consacrées à la formation des élu·es local·es pour les exercices 2020 à 2025.

## Contenu

- `index.html` : portail public des cartes
- `communes.html` : carte des communes
- `intercommunalites.html` : carte des EPCI
- `departements.html` : carte des départements et CTU
- `regions.html` : carte des régions

## Publication avec GitHub Pages

1. Créer un dépôt GitHub public.
2. Déposer tous les fichiers de ce dossier à la racine de la branche `main`.
3. Dans **Settings > Pages**, choisir **Deploy from a branch**.
4. Sélectionner la branche `main` et le dossier `/ (root)`.
5. Enregistrer.

Le site sera ensuite disponible à une adresse de la forme :

`https://NOM-UTILISATEUR.github.io/NOM-DU-DEPOT/`
